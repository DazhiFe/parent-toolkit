# 爸妈工具箱 — 功能规划建议

## 现状分析

项目已有 22 个页面/功能，覆盖数学练习、语文工具、英语拼读、图片处理、PDF工具、AI导航等。但存在以下明显缺口：

| 领域 | 现状 | 缺口 |
|---|---|---|
| 语文 | 古诗词 + 拼音标注 | **缺生字练习/字帖**（小学最刚需） |
| 英语 | 单词拼读 | **单词记忆为占位符**，缺词汇积累工具 |
| 数学 | 5个出题器 + 百数表 | 缺**口算计时挑战**（趣味性不足） |
| 效率 | 番茄钟 | 缺**课程表/习惯打卡**等日常管理工具 |
| 文档 | PDF编辑/转PPT | **Word转PDF代码已写但未注册路由** |
| AI推荐 | 16项全为占位符 | 无实际功能 |

---

## 建议新增功能（按优先级排序）

### P0 — 刚需高频（小学家长日常必用）

#### 1. 生字字帖生成器 `/character-practice`
- **功能**：输入或选择生字（支持按年级/课文筛选），生成可打印的字帖，含田字格、笔顺提示、描红区、自由书写区
- **为什么做**：小学1-3年级家长几乎每天需要给孩子准备字帖练习，目前只能买字帖本或手动画格
- **实现方式**：纯前端，SVG 绘制田字格 + Canvas 生成图片/PDF，笔顺数据可用开源 hanzi-writer
- **参考**：汉典字帖、字帖生成器

#### 2. 英语单词闪卡 `/word-flashcard`
- **功能**：按年级/主题分类的单词闪卡，支持翻卡查看释义和发音，内置艾宾浩斯遗忘曲线复习提醒（localStorage 存储）
- **为什么做**：替换当前占位符"单词记忆"，小学英语词汇积累是家长高频需求
- **实现方式**：纯前端，Web Speech API 发音 + localStorage 存储学习进度
- **数据**：内置小学1-6年级核心词汇（约600词），JSON 外置按需加载

#### 3. 口算计时挑战 `/math-sprint`
- **功能**：限时口算挑战模式（如60秒内做对尽可能多的题），支持难度选择，实时计分+正确率，历史最高分记录
- **为什么做**：现有出题器只能打印，缺少互动趣味性；口算速度是小学数学核心能力，计时挑战能激发孩子积极性
- **实现方式**：纯前端，复用现有出题逻辑 + 计时器 + localStorage 排行榜

### P1 — 实用工具（提升日常效率）

#### 4. 课程表生成器 `/schedule-maker`
- **功能**：可视化编辑每周课程表，支持自定义科目/颜色/时间段，一键生成可打印的课程表（A4排版）
- **为什么做**：每学期开学家长都需要帮孩子做课程表，目前只能用Word手动排版
- **实现方式**：纯前端，拖拽/点击编辑 + html2canvas 生成图片或 jsPDF 生成 PDF

#### 5. 习惯打卡 `/habit-tracker`
- **功能**：自定义每日习惯项（如：阅读30分钟、练字1页、跳绳100下），孩子每天打卡，连续打卡有星星奖励，周/月统计
- **为什么做**：培养孩子好习惯是家长长期痛点，打卡+奖励机制比口头督促有效
- **实现方式**：纯前端，localStorage 存储，日历视图展示打卡记录

#### 6. 亲子阅读记录 `/reading-log`
- **功能**：记录每天陪孩子读的书名、时长、简单感想，自动统计月度阅读量，可生成阅读报告
- **为什么做**：学校常要求家长提交阅读记录，目前只能手写或用Excel
- **实现方式**：纯前端，localStorage 存储，可导出为图片/PDF

### P2 — 补全现有功能

#### 7. 注册 Word 转 PDF 路由
- **现状**：`WordToPdfView.vue` 代码已写好，但路由未注册
- **操作**：在 router/index.js 添加路由，在首页/相关页面添加入口

#### 8. 成语学习工具 `/idiom-learning`
- **功能**：按年级分类的成语卡片（释义+例句+近义/反义），支持搜索和收藏，可生成成语填空练习题
- **为什么做**：成语是小学语文考试必考项，目前缺专门工具

#### 9. 数学乘法口诀表 `/multiplication-table`
- **功能**：交互式九九乘法表，点击任意格可听发音，支持随机出题练习模式，可打印
- **为什么做**：二年级必背乘法口诀，交互式比纸质更有效

## 实施建议

### 第一批（3个，ROI 最高）
1. **生字字帖生成器** — 刚需最强，差异化明显
2. **口算计时挑战** — 复用现有代码，开发量小
3. **注册 Word 转 PDF** — 5分钟搞定

### 第二批（3个）
4. **英语单词闪卡** — 补全占位符
5. **课程表生成器** — 开学季刚需
6. **乘法口诀表** — 开发量小

### 第三批
7. **习惯打卡** — 长期留存功能
8. **亲子阅读记录** — 配合学校要求
9. **成语学习** — 语文补充

---

## 技术要点

- 所有新功能均为**纯前端实现**，无需后端
- 词汇/成语等大数据采用 JSON 外置 + 按需加载（参考 poems.json 模式）
- 生字笔顺数据依赖 `hanzi-writer`（约 50KB gzip），按需加载
- 所有工具支持**打印/PDF导出**（复用现有 jsPDF/html2canvas）
- 移动端优先适配（家长多用手机）
