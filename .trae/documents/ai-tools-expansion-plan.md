## 实施方案

### 修改文件：`src/data/aiTools.js`
在 `aiTools` 数组中按分类追加以上 13 个工具对象，字段结构与现有一致：
```js
{ id: 'kimi', name: 'Kimi', url: 'https://kimi.moonshot.cn',
  description: '月之暗面出品，超长文本处理能力最强，免费',
  category: 'chat', color: 'purple', letter: 'K', tag: '热门' },
```

- 高价值工具标记 `tag: '热门'`（Kimi、可灵AI、Suno）
- 颜色分配避免同分类内重复
- letter 优先用工具名首字（中文取首字，英文取首字母）

### 新增工具总数
- 对话: 5 → 8
- 资讯: 2 → 3
- 图形: 1 → 4
- 视频: 1 → 3
- 学习: 2 → 3
- 音频: 1 → 4
- **总计: 12 → 25**

### 无需修改的文件
- `AiToolsView.vue` — 已是数据驱动，自动渲染新增工具
- `SideNav.vue` — 分类锚点不变，无需改动
- `useAiToolClicks.js` — localStorage 以 tool.id 为 key，新工具自动支持

## 验证步骤
1. `npm run dev` 本地预览，检查 6 个分类下工具数量和展示
2. 验证新工具的链接可正常跳转
3. 验证点击统计、隐藏、失效标记对新工具生效
4. `npm run build` 确认构建通过
