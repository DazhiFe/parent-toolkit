# 家长工具箱

一个为家长精心打造的AI工具导航和育儿资源集合平台，帮助家长更好地辅助孩子成长。

## 项目特点

- 精选AI工具导航
- 学习资源整合
- 成长工具集合
- 响应式设计，适配各种设备
- 现代化UI，基于Tailwind CSS

## 技术栈

- Vue 3
- Vite
- Vue Router
- Pinia
- Tailwind CSS
- Axios

## 开发环境设置

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

### 代码检查

```bash
npm run lint
```

## 项目结构

```
parent-toolkit/
├── public/                 # 静态资源
├── src/
│   ├── assets/            # 资源文件
│   │   └── styles/        # 样式文件
│   ├── components/        # 公共组件
│   ├── router/            # 路由配置
│   ├── views/             # 页面组件
│   ├── App.vue            # 根组件
│   └── main.js            # 入口文件
├── index.html             # HTML模板
├── package.json           # 项目配置
├── vite.config.js         # Vite配置
├── tailwind.config.js     # Tailwind配置
└── README.md              # 项目说明
```

## 功能模块

- 首页：展示网站介绍和热门工具
- AI工具：AI工具导航和分类
- 学习工具：学习辅助工具集合
- 成长工具：孩子成长相关工具
- 关于：项目介绍和联系方式
